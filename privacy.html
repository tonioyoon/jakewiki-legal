개인정보처리방침

시행일: 2026년 5월 25일 · 앱: JakeWiki (com.jakefamily.jakewiki)

JakeWiki는 개인정보를 수집하지 않습니다. 계정도, 서버도, 분석 도구도 없습니다. 모든 자료와 AI 처리는 전적으로 사용자의 기기 안에서 이루어집니다.

1. 우리가 수집하지 않는 것

JakeWiki는 다음을 일절 수집·전송·저장하지 않습니다.

  • 이름, 이메일, 전화번호 등 식별 정보
  • 계정 정보 (로그인·회원가입 자체가 없습니다)
  • 광고 식별자, 위치 정보, 사용 통계, 크래시 리포트
  • 사용자가 입력하거나 정리한 위키 내용

JakeWiki에는 개발자가 운영하는 서버가 없습니다. 따라서 사용자의 데이터가 개발자에게 전송되는 경로 자체가 존재하지 않습니다.

2. 기기에 저장되는 데이터

사용자가 앱에 던진 자료(텍스트·링크 등), 그로부터 생성된 위키 페이지, 다운로드한 AI 모델 파일은 모두 사용자 기기에만 저장됩니다.

  • 기본 위치는 앱 내부 저장소입니다.
  • 설정에서 사용자가 직접 외부 폴더를 지정할 수 있습니다(예: Obsidian 동기화 폴더). 이 경우 데이터는 사용자가 선택한 폴더와 그에 연결된 동기화 서비스의 통제 하에 놓입니다.
  • 앱을 삭제하면 내부 저장소의 데이터도 함께 삭제됩니다.

3. 네트워크 통신

JakeWiki가 인터넷에 연결하는 경우는 다음 두 가지뿐입니다.

3.1 AI 모델 다운로드

최초 1회, 온디바이스 LLM 모델 파일을 Hugging Face에서 내려받습니다. 이 과정에서 개인정보는 전송되지 않으며, 통상적인 파일 다운로드 요청만 발생합니다.

3.2 사용자가 요청한 URL 가져오기

사용자가 링크를 던지면, 그 링크의 본문을 가져와 위키로 정리하기 위해 해당 웹사이트에 직접 요청을 보냅니다. 이 요청은 사용자가 명시적으로 링크를 추가했을 때만 발생하며, 가져온 내용은 기기 안에서 처리됩니다.

4. 제3자 서비스

위 통신의 상대방(Hugging Face, 사용자가 입력한 웹사이트의 운영 주체)은 JakeWiki와 무관한 제3자이며, 각자의 개인정보처리방침을 따릅니다. JakeWiki는 광고·분석 SDK를 일절 포함하지 않습니다.

5. 아동의 개인정보

JakeWiki는 개인정보를 수집하지 않으므로 아동을 포함한 누구의 개인정보도 수집되지 않습니다.

6. 권한 사용

  • 인터넷 — 위 3항의 모델 다운로드 및 URL 가져오기
  • 알림 — 모델 다운로드·자료 처리 완료 안내
  • 포그라운드 서비스 — 대용량 모델 다운로드를 백그라운드에서 안정적으로 진행

7. 방침 변경

본 방침이 변경되면 이 페이지가 갱신되며 상단의 시행일이 함께 수정됩니다.
